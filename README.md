# Student Expense Tracker (Java Swing)

A desktop application built using Java Swing that allows students to track and manage their daily expenses.

## Features
- Add expenses
- Delete expenses
- View expenses in a table
- Calculate total spending
- View category-wise totals
- File storage (data persistence)

## Technologies Used
- Java
- Swing (JFrame, JTable, JOptionPane)
- File Handling
- Object-Oriented Programming

## How to Run
1. Open project in NetBeans
2. Run `ExpenseTrackerGUI.java`

## Future Improvements
- Add charts (pie/bar graphs)
- Add login system
- Export data to PDF
- Database integration (MySQL/PostgreSQL)
